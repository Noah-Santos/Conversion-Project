Binary

Binary converts the number into a base 2 form. It uses 1s and 0s to create numbers. It reads right to left and goes in 2 to the power. It starts at the poiwer of 0 and then the power increases by 1.







Hexadecimal

Hexadecimal converts numbers in a base 16. It goes from 1-15. It uses 1-9 normally, but 10-15 are represented by A-F. To convert, you divide the number by 16. The remainder is used to create the result. You divide how many times the number is able to be completely divided by 16 by 16 again and the remainder is used. The cycle repeats until the number is no longer divsible by 16.
